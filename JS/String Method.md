# String Method

### repeat()

`repeat()` 메서드는 호출된 문자열의 지정된 `count`의 복사본이 있는 새 문자열을 반환한다.

```javascript
let str = 'Hello World ';
cnosole.log(str.repeat(2))
// Hello World Hello World
```



### charAt()

`charAt()` 메서드는 문자열의 지정된 인덱스에 있는 문자를 반환한다.

※ 마지막 문자의 인덱스는 string.length-1

```javascript
let str = "HELLO WORLD";
let res = str.charAt(0);
console.log(res); // H
```
