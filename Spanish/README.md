# 스페인어 입력 방법

### 스페인어 전환방법 

**: Shift + Alt**



| 특수문자      | 입력방법          |
| ------------- | ----------------- |
| á, é, í, ó, ú | ´ + a, e, i, o, u |
| ñ             | 세미콜론(;) 입력  |
| ¿             | Shift + =         |
| ?             | Shift + -         |
| ¡             | =                 |
| !             | Shift + 1         |
